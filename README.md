# The-beginner---HANGMAN-Game-Python-
🎮 Hangman Game

A simple and interactive Hangman game built to test your word-guessing skills. The player must guess a hidden word one letter at a time before running out of chances.

🧠 How It Works
A random word is selected and displayed as blank spaces.
The player guesses letters one by one.
✅ Correct guesses reveal the letter in the word.
❌ Incorrect guesses reduce the remaining attempts and draw the hangman figure.
The game continues until:
The player guesses the word (Win 🎉), or
The hangman is fully drawn (Game Over 💀)
